# 🚗 AutoGuard OS – Vehicle Fault Detection & Risk Prioritization System

AutoGuard OS is a real-time vehicle monitoring and intelligent alert prioritization system built using Flask and SQLite.

## 🔥 Features
- Real-time vehicle data simulation
- Risk scoring algorithm (Probability × Severity × Impact)
- Alert prioritization (Low / Medium / Critical)
- Persistent database storage
- Live dashboard UI
- Activity logging system

## 🧠 Risk Model
Risk Score = Probability × Severity × Impact

## ⚙️ Tech Stack
- Python (Flask)
- SQLite
- HTML + TailwindCSS
- JavaScript

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run server:
   python app.py

3. Open:
   http://127.0.0.1:5000
